# Ex01 — Solução

Arquivo:

```text
stack_sentinel/clients/mock_service_client.py
```

Substitua a função `check_mock_service_health` por:

```python
def check_mock_service_health(client: Optional[MockServiceClient] = None) -> bool:
    client = client or MockServiceClient()
    response = client.get_json(MOCK_ENDPOINTS["health"])
    return response.get("ok") is True
```

## Observação sobre a FastAPI

A mock API agora é um serviço FastAPI real. Essa solução usa `MockServiceClient` porque ele já encapsula a chamada HTTP para o endpoint `/health`.

Também seria válido chamar o endpoint diretamente com `urllib.request.urlopen`, desde que a função continue retornando apenas `True` ou `False` e não quebre quando o serviço estiver indisponível.

Exemplo alternativo:

```python
def check_mock_service_health(client: Optional[MockServiceClient] = None) -> bool:
    client = client or MockServiceClient()
    try:
        response = client.get_json("/health")
    except Exception:
        return False
    return response.get("ok") is True
```
