# Ex02 — Solução

Arquivo:

```text
stack_sentinel/mcp_server/tools.py
```

Substitua a função `fetch_ticket_context` por:

```python
def fetch_ticket_context(ticket_id: str, client: Optional[MockServiceClient] = None) -> Dict[str, Any]:
    if not ticket_id:
        return {"ok": False, "error": "ticket_id is required"}

    client = client or MockServiceClient()
    response = client.get_ticket(ticket_id)

    if not response.get("ok"):
        return {"ok": False, "error": response.get("error", "ticket not found")}

    data = response.get("data") or {}
    return {
        "ok": True,
        "id": data.get("id"),
        "summary": data.get("summary"),
        "severity": data.get("severity"),
        "service": data.get("service"),
        "status": data.get("status"),
        "build_id": data.get("build_id"),
    }
```

## Solução alternativa válida

Como o serviço mockado expõe `GET /tickets/{ticket_id}` via FastAPI, também é aceitável buscar o ticket usando `client.get_json(...)` diretamente, em vez de `client.get_ticket(...)`.

```python
def fetch_ticket_context(ticket_id: str, client: Optional[MockServiceClient] = None) -> Dict[str, Any]:
    if not ticket_id:
        return {"ok": False, "error": "ticket_id is required"}

    client = client or MockServiceClient()
    response = client.get_json(MOCK_ENDPOINTS["tickets"].format(ticket_id=ticket_id))

    if not response.get("ok"):
        return {"ok": False, "error": response.get("error", "ticket not found")}

    data = response.get("data") or {}
    return {
        "ok": True,
        "id": data.get("id"),
        "summary": data.get("summary"),
        "severity": data.get("severity"),
        "service": data.get("service"),
        "status": data.get("status"),
        "build_id": data.get("build_id"),
    }
```

Para essa alternativa, garanta que `MOCK_ENDPOINTS` esteja importado no arquivo:

```python
from stack_sentinel.clients.mock_service_client import MOCK_ENDPOINTS, MockServiceClient
```
