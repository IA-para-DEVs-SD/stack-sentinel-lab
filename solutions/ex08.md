def compile_minimal_graph() -> SimpleGraph:
    graph = SimpleGraph()

    def echo_node(state: AgentState) -> AgentState:
        updated = dict(state)
        updated["final_answer"] = state.get("user_input")

## Desafios opcionais — solucao proposta

Uma versão com nomes pode armazenar tuplas:

```python
class SimpleGraph:
    def __init__(self):
        self.nodes = []

    def add_node(self, node, name=None):
        self.nodes.append((name or node.__name__, node))

    def run(self, state):
        current = state
        for _, node in self.nodes:
            current = node(current)
        return current
```

history = list(current.get("history") or [])
