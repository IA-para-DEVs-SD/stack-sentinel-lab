# Ex11 — Solução

Arquivo:

```text
stack_sentinel/agent/graph.py
```

Substitua a função `route_by_intent` por:

```python
def route_by_intent(state: AgentState) -> str:
    routes = {
        "ticket": "fetch_ticket",
        "build": "fetch_build",
        "docs": "fetch_docs",
    }
    return routes.get(state.get("intent"), "fallback")
```

## Desafios opcionais — solucao proposta

Uma rota composta pode ser representada por uma string nova:

```python
if state.get("ticket_id") and state.get("build_id"):
    return "fetch_ticket_and_build"
```

Para justificar a decisão sem quebrar o contrato testado, crie uma segunda função:

```python
def explain_route_by_intent(state: AgentState) -> dict:
    route = route_by_intent(state)
    return {"route": route, "reason": f"intent={state.get('intent')}"}
```
